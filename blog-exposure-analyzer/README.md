## 🚀 빠른 시작

### 1. 환경 변수 설정

```bash
cp .env.example .env
# .env 파일을 열어서 실제 값 입력
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 데이터베이스 마이그레이션

```bash
npm run db:push
```

### 4. 서버 시작

```bash
# 백엔드 서버 (포트 3000)
npm run server

# 프론트엔드 (포트 5173)
npm run dev
```

### 5. 브라우저에서 접속

```
http://localhost:5173
```
